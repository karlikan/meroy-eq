# Инструкция по сборке и установке Meroy EQ

Этот проект использует фреймворк **JUCE** и систему сборки **CMake**.

## 1. Подготовка
Перед сборкой убедитесь, что у вас установлены:
- **Visual Studio 2022** (с компонентом "Разработка классических приложений на C++").
- **CMake** (версии 3.22 или выше).
- **Git** (для скачивания зависимостей JUCE).

## 2. Сборка проекта
1. Откройте терминал (PowerShell или CMD) в папке проекта `D:\НОВЕЙШАЯ ВСЯЧИНА\Vibecode\vst_eq`.
2. Создайте папку для сборки и перейдите в неё:
   ```powershell
   mkdir build
   cd build
   ```
3. Сконфигурируйте проект с помощью CMake:
   ```powershell
   cmake ..
   ```
   *На этом этапе CMake автоматически скачает JUCE. Это может занять пару минут.*
4. Скомпилируйте плагин:
   ```powershell
   cmake --build . --config Release
   ```

## 3. Установка плагина
После успешной компиляции готовый файл плагина будет находиться по пути:
`D:\НОВЕЙШАЯ ВСЯЧИНА\Vibecode\vst_eq\build\MeroyEQ_artefacts\Release\VST3\Meroy EQ.vst3`

**Чтобы пользоваться плагином:**
1. Скопируйте папку (или файл) `Meroy EQ.vst3`.
2. Перейдите в вашу личную папку с плагинами (которую вы указали в DAW).
3. Вставьте туда скопированный файл.
4. Откройте вашу DAW (Ableton, FL Studio, Reaper и т.д.) и обновите список плагинов.

## 4. Особенности Meroy EQ
- **Частотный диапазон:** до 30 КГц.
- **Интерфейс:** 60 FPS Spectrum Analyzer на фоне.
- **Цвета:** Премиальный темно-серый и золото.
- **Управление:** Мышь для перемещения узлов, колесо мыши для изменения добротности (Q).
